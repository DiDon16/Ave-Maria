Plateforme de Gestion Sécurisée des Analyses Médicales

Ce projet est une plateforme minimaliste permettant aux docteurs d’ajouter des patients, de faire des analyses, et d’obtenir des prédictions du stade de la maladie rénale chronique (MRC), tout en mettant un accent fort sur la sûreté des accès et la confidentialité des données.

Fonctionnalités principales

  • Inscription et validation manuelle des comptes docteurs
  • Double authentification par code OTP (5 chiffres)
  • Dashboard pour gérer patients et analyses
  • Connexion à une API de prédiction pour la MRC
  • Rôles : Médecins (lecture + écriture), Infirmières (lecture seule)

Prérequis

  • PHP 8.1+
  • Composer
  • MySQL
  • Git
  • Un serveur local : XAMPP, MAMP ou Laragon

Installation

1. Cloner le projet principal (ave-maria)
git clone https://github.com/DiDon16/Ave-Maria.git
cd Ave-Maria
2. Installer les dépendances
composer install
3. Créer le fichier .env et la clé de l'application
cp .env.example .env
php artisan key:generate
4. Configuration de la base de données (.env)
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=nom_de_la_base
DB_USERNAME=root
DB_PASSWORD=
5. Exécuter les migrations et les seeders
php artisan migrate --seed
6. Lancer le serveur
php artisan serve
7. Cloner le projet de l'API de prediction (brief-prediction-api)
Dans un autre terminal et à lextérieur du dossier Ave-Maria 
git clone https://github.com/DiDon16/brief-prediction-api.git
cd brief-prediction-api
8. Installer les dépendances
composer install
9. Créer le fichier .env et la clé de l'application
cp .env.example .env
php artisan key:generate
10. Lancer le serveur

Sur un port different du port 8000

php artisan serve --port=numero-port
11. Configurer l'application pour communiquer avec l'API

Se rendre dans le fichier .env de Ave-Maria et rechercher la ligne : 

MRC_API_URL=

Ajouter l'URL de l'API , ce qui donne : 

MRC_API_URL=http://127.0.0.1:numero-port/api/predict

Accès à l'appication Web ave-maria : http://127.0.0.1:8000

Accès à la documentation de l'API : http://127.0.0.1:numero-port/api/documentation

Configuration des emails (OTP)

Utilisez Mailtrap ou Gmail dans le fichier .env :

MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=utilisateur
MAIL_PASSWORD=motdepasse
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="noreply@example.com"
MAIL_FROM_NAME="AppName"

Double authentification (2FA)

  • Connexion avec mot de passe
  • Génération d'un code OTP (5 chiffres)
  • Envoi du code par email et validation

Technologies

  • Laravel 10+
  • MySQL
  • Bootstrap 5
  • API externe pour prédiction MRC (simple simulation)
